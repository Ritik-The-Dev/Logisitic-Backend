# Logistic

## Stack Used
```
Node JS (Express), MongoDB
```

## Node & NPM Version
```
v18.20.5 & 10.9.2
```

## How to run the application
```
Step 1 => Install node modules - npm install

# For development
-----------------
Step 2 => Start the application
    Step 2.1 => npm run dev

# For Production
-----------------
Step 2 => Firstly run
    Step 2.1 => npm run build 
    Step 2.2 => npm start
```
